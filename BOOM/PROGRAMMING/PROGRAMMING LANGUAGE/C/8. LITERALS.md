
## Integer literals 

Prefixes in Integer literals

No prefix -> Decimal 
     0x  -> Hexadecimal
     0    -> Octal
     0b  -> Binary(Not standard. ,but supported by compiler like GCC )
      

```c
#include<stdio.h>
int main()
{
   int a =20;    //20
   int b =0x1A;  //26   Hexadecimal
   int c =016;   //14   Octal
   int d =0b11;  //3    Binary
   printf("%d %d %d %d %d",a,b,c,d);
}
```


Suffixes in Integer Literals

```c
#include<stdio.h>
int main()
{
   int a=124;
   unsigned int b = 1244;
   long int c = 124l;
   long long d = 124ll;
   return 0;
}
```


## Floating point literals

Allowed prefix: 0x
Allowed suffixes:
     f : float
     l : long double


```c
#include<atdio.h>
int main()
{
    float a=10.5f;
    double b=10.515;
    long double c=2.1e4f;
    float e=200.1e-80;
    double f=0x1A.1p2;
    printf("%g %g %g %g %g %g",a,b,c,d,e,f);
    return 0; 
}
```

Output:
10.5
10.515
10.515
2100
2.001e-78
104.25


## Character and string literals

```c
#include<stdio.h>
int main()
{
    char c='g';
    const char *s="g+g";
    print("%c %s",c,s);
    return 0;
}
```

Output:
g
g+g
