
Each data types have the format specifiers.


```c
#include<stdio.h>
int main()
{
    int age=39;
    char name[]="Sandeep";
    float weight = 68.5;
    bool isMarried = true;
    char gender = 'M';
    printf("%d\n",age);
    printf("%s\n",name);
    printf("%f\n",weight);
    printf("%d\n",married);
    printf("%c",gender);
    return 0;
}
```


## PRIMTIVE TYPES:

Integer types :
    short, int ,long , long long , unsigned short , 
    unsigned int, unsigned long , unsigned long long

Character types:
     char , unsigned char

Floating types: 
     float, double, long double

Others: 
     bool, void


## NON PRIMITIVE TYPES:

Arrays, Pointers , User Defined(struct)
