
 By default global variables are defined by zero.



```c
#include<stdio.h>
int main()
{
    int x=10;    // local variable
    print("%d",x);
    return 0;
}
```


```c
#include<stdio.h>
int main()
int x=10;  // global variable
{}
    print("%d",x);
    return 0;
}
```


what happens when we have local and global variables with the same name?

```c
#include<stdio.h>
int main()
int x=10;  // global variable
{   
    int x=20;
    print("%d",x);
    return 0;
}
```

Output:   20
