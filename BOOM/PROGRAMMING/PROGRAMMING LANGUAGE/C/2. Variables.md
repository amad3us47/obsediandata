Variables allow you to operate on data.
Variables are label to those those memory address block.
We can modify the variables.
Give variable a meaningful name.
It has a data type.

```c
#include<stdio.h>
int main()
{
   int x=5;
   printf("%d",x);
   int y=10;
   printf("%d\n",y);
   y=20;
   printf("%d",y);
   return 0;
}
```
## Variable naming rules

Allowed character:
- Lower case('a' to 'z')
- Upper case('A' to 'Z')
- Digits ('0' to '9')
- Underscore (' _ ')
Cannot begin with a digit. 
Cannot be a keyword.
