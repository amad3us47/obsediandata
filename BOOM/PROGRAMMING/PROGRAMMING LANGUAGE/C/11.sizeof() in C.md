An operator in c
Return number of bytes required for a data type
Can be used with variables and literals also
Operand's size is evaluated at compile time


```c
#inlcude<stdio.h>
int main()
{
     int x=10;
     double d=11.5;
     printf("%zu\n",x);
     printf("%zu\n",d);
     printf("%zu\n",15ll);
     printf("%zu\n",13.6f);
     printf("%zu\n",x+10);
     return 0;
}
```

4
