An operator in C
Return number of bytes required for a data type
Can be used with variables and literals also
Operand's size is evaluated at compile time

```c
#include<stdio.h>
int main()
{
  int x=10;
  double d=11.5;
  printf("%zu\n",x);
  printf("%zu\n",d);
  printf("%zu\n",1511);
  printf("%zu\n",x+10;)
  return 0;  
}
```

Output:
4
8

4

```c
#include<stdio.h>
int main()
{
   int x=10;
   printf("%zu\n",sizeof(x++));
   printf("%d",x);
   return 0;
}
```

Output:
4
10
