Basic usage :-
```
gospider -q -s "https://www.example.com" -o output
```


Dept search :-
```
gospider -q -s "https://www.example.com/products" -d 3 -o output
```


```
gospider -q -s "https://www.example.com/products" -d 3 -o output -d 0 -p http://127.0.0.1:8080
```
