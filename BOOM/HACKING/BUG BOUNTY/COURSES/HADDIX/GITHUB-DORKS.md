
Installation :-
```
git clone https://github.com/techgaun/github-dorks.git
```

```
cd github-dorks
```

```
pip3 install -r requirements.txt
```



Search a repo :-
```
github-dork.py -r techgaun/github-dorks   
```

Search all repo of a user :-
```
github-dork.py -u techgaun
```

Search all of repos of an organization :-
```
github-dork.py -u dev-nepal
```
